## 🐶 소개 
- 내가 키우는 반려동물이 SNS을 한다면?
- 반려인들의 유머러스한 반려일상 공유 공간을 제공하는 SNS
- 최근 온라인에서 화제가 되는 트렌드인 `부캐` (특정 컨셉을 적용한 일종의 가면놀이 컨텐츠)와 `반려 동물` 을 결합한 프로젝트
